## 設計語言

- **質感**：未來感 + 高端百貨；金色霓虹 + 寶山藍夜空 + 晶圓格紋
- **字型**：粗黑體中文 + Geometric Sans（Geist / Inter）英文
- **氛圍**：科技感（晶圓、奈米、EUV）+ 高端購物（金、燈光、霓虹）+ 台中地景（歌劇院曲線）
- **禁用**：俗氣的 emoji 大集合、過多漸層色、cartoonish 風格

## 主要視覺元件

1. **Hero Cover**：登入頁 + 排行榜頂部 banner
2. **Simon 寶山玄彬 Mascot**：吉祥物 / 立牌 / 各種情緒貼紙
3. **樓層導覽圖**：B1~6F 七層的縮圖
4. **遊戲徽章**：12 款小遊戲 + 5 款料理 + 3 款合照 = 20 個徽章
5. **緊急任務徽章**：寶山召集令 / 玄彬密語 / 漢神尋寶
6. **完賽勳章**：Top 1/2/3 + 全勤 + 隊伍合作
